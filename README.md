# React E-Commerce App

Bu proje, React öğrenmek ve pratik yapmak amacıyla geliştirilmiş bir e-ticaret uygulamasıdır.  
Amaç, React ekosistemindeki temel kavramları ve modern bir frontend projesinin temel yapı taşlarını (Routing, API iletişimi, State Management, UI Design) uygulamalı olarak öğrenmektir.

## Kullanılan Teknolojiler
- React (Vite altyapısı ile)
- TypeScript
- Redux Toolkit (State Yönetimi)
- React Router DOM (Sayfa Yönlendirmeleri)
- Material UI (MUI) (Arayüz Bileşenleri)
- React Toastify (Bildirimler)
- Axios (API İstekleri)
- CSS

## Özellikler
- Ürün Listeleme: API'den gelen verilerin listelenmesi.
- Ürün Detay: Seçilen ürünün detay sayfasına dinamik yönlendirme.
- Arama ve Filtreleme: Navbar üzerinden ürün ismine göre anlık arama yapabilme.
- Sepet Yönetimi: Ürünleri sepete ekleme ve adet belirleme, sepet içeriğini sağ panelde (Drawer) görüntüleme, sepetten ürün silme, LocalStorage Entegrasyonu: Sayfa yenilense dahi sepet verilerinin korunması.
- Satın Alma Simülasyonu: Kullanıcı bakiyesinin kontrol edilmesi, yetersiz bakiye uyarısı ve satın alma sonrası bakiyenin güncellenmesi.
- Kullanıcı Girişi: Login ve Register arayüz tasarımları.
