# React E-Commerce App

Bu proje, React öğrenmek ve pratik yapmak amacıyla geliştirilmiş bir e-ticaret uygulamasıdır.  
Amaç, React ekosistemindeki temel kavramları uygulamalı olarak öğrenmek ve zamanla gerçekçi bir proje yapısına dönüştürmektir.

## Proje Amacı
- React temel kavramlarını öğrenmek (component, props, state vb.)
- Modern frontend geliştirme pratiği kazanmak
- E-ticaret senaryosu üzerinden gerçek hayata yakın örnekler geliştirmek
